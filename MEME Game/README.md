# Strategy Game

## How to Run

1. **Download Godot Engine**: Ensure you have Godot Engine 4.x installed. You can download it from [godotengine.org](https://godotengine.org/download).
2. **Open the Project**:
   - Launch Godot Engine.
   - Click **Import** and select the `project.godot` file in this folder.
   - Alternatively, dragging the `project.godot` file onto the Godot executable might works.
3. **Play**:
   - Once the project is open in the editor, press the **Play** button in the top-right corner (or press `F5`).
   - The game should launch directly into the Main Menu.

## Controls
- **Mouse**: Select units and place them on the grid.
